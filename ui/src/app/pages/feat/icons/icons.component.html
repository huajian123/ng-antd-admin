图标选择器

内置图标选择器，支持关键词搜索与前端分页，选中后同步到输入框。
线框风格

ng-zorro 内置的 outline 主题图标，通过 nzType 指定图标名，nzTheme 指定风格（outline / fill / twotone）。

@for (icon of outlineIcons; track icon.type) {
{{ icon.label }}
}
双色风格

双色图标支持通过 nzTwotoneColor 自定义强调色。

@for (icon of twotoneIcons; track icon.type) {
{{ icon.label }}
}
自定义 SVG 图标

在 i[nz-icon] 内直接放 svg 元素，或通过 provideNzIcons 全局注册。

阿里图标库

去 iconfont.cn 生成在线 JS，通过 LoadAliIconCdnService 在应用启动时加载，模板中用 nzIconfont 渲染。
