input()

父组件修改 signal，子组件通过 input() 接收，computed() 自动派生大写值。

子组件 <app-child-display>：

// 父组件
readonly parentText = signal('Hello');

// 子组件
readonly value = input.required<string>();
readonly upperValue = computed(() => this.value().toUpperCase());
output()

子组件点击按钮触发 output()，父组件接收事件更新自身 signal。

子组件 <app-child-counter>（仅计数部分）：

父组件接收到的计数： {{ receivedCount() }} 
// 子组件
readonly countChange = output<number>();
increment(): void {{ '{' }}
          this.count.update(v => v + 1);
  this.countChange.emit(this.count());
          {{ '}' }}

          // 父组件模板
<app-child-counter (countChange)="onCountChange($event)" />
model()

model() 是 input + output 的语法糖，支持 [(xxx)] 双向绑定语法。

父组件状态： {{ parentChecked() ? '开启' : '关闭' }}

子组件（开关在子组件内）：

// 子组件
readonly checked = model(false);

// 父组件模板
<app-child [(checked)]="parentChecked" />
// 等价于：
// [checked]="parentChecked()"
// (checkedChange)="parentChecked.set($event)"
