交错进场

通过 CSS animation-delay 配合 @for 的 $index 实现交错动画，无需任何动画库

@for (item of items(); track item) {
{{ $index + 1 }}
}

核心原理

每个元素的 animation-delay = $index × 间隔时间，配合 @if / @for 控制元素的添加即可实现交错效果。

@for (item of items(); track item) {
  <div
    class="stagger-item"
    [style.animation-delay]="$index * 80 + 'ms'"
  >{{ '{{' }} item {{ '}}' }}</div>
}
